# API Map

The generated Rust API documentation is built by:

```bash
bash scripts/build_docs_site.sh
```

After the site is built, open:

```text
target/docs-site/api/index.html
```

Important public modules and items:

- `tidu::Primitive`
- `tidu::PrimitiveBuilder`
- `tidu::PrimitiveValue`
- `tidu::ADKey`
- `tidu::linearize`
- `tidu::linear_transpose`
- `tidu::linear_transpose_with_builder`
- `tidu::LinearizedGraph`
- `tidu::PrimitiveGraph`
- `tidu::eager`
